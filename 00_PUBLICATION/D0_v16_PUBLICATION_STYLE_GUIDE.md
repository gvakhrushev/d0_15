# D0 v16 Publication Style Guide

Status: PUBLICATION-STYLE-GUIDE / EDITORIAL-GUARDRAIL

## 1. Claim verbs
Use **proves** only for finite theorem/certificate-owned claims. Use **supports**, **models**, **bridges**, **suggests**, or **tests** for external laboratory, observational, or numerical material.

## 2. External bridges
Laboratory analogues, LIGO/GWOSC analyses, PDG comparisons, cosmology survey fits, and dusty-plasma protocols must be labelled as bridge/passport/external-test material unless a specific internal theorem owner and executable certificate are named.

## 3. Formula ownership
Every high-sensitivity formula should state whether it is:

```text
internal theorem / executable certificate / bridge dictionary / empirical passport / prediction target / no-go
```

## 4. Publication status vocabulary
Allowed external-facing labels:

```text
CORE THEOREM
SECTOR THEOREM
OPERATOR SCAFFOLD
BRIDGE LAW
EMPIRICAL PASSPORT
NO-GO THEOREM
PREDICTION TARGET
LAB-BRIDGE / TABLETOP-PASSPORT-SEED
NEGATIVE CONTROL
```

Forbidden promotional shortcuts:

```text
external analogue proves core physics
single-event scan proves discovery
fixed numerical match proves theorem
laboratory bridge equals cosmological proof
```

## 5. Reader-facing order
The publication reading order is:

```text
Book 00 -> Book 01 -> Book 02 -> Book 03 -> Book 04 -> Book 05 -> Book 06 -> Book 07 -> Book 08
```

Historical section numbers may be preserved when cross-references depend on them. In that case, the local text order is authoritative.
