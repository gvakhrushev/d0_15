# Book 00 Process Appendix

This appendix preserves process and audit material that should not dominate the entry contract. The canonical books remain theorem/operator/certificate/bridge first; this file records how working rules are applied to the corpus.

## Process Recursion

D0's corpus process preserves the proof trail because the theory treats finite registration, comparison and return as primitive. Failed paths, negative controls and retired shortcuts remain useful as diagnostic archive material.

```math
D0_{n+1}=\operatorname{integrate}\!\left(\operatorname{readout}\bigl(\operatorname{test}(\operatorname{synthesize}(D0_n+\text{new material}))\bigr)\right).
```

Reports are not results. A report becomes part of the active corpus only after it changes a theorem, proof, certificate, guardrail, or bridge boundary. Closure precedes status. If a proposed expression requires a new untyped anchor, a free fit, or external target feedback, it is not a D0 closure.

## Critic-Response Closure Rule

A reviewer attack is treated as a proof-cell request. If a critic points to a naked exponent, hidden Standard-Model bridge, untyped external carrier, missing data manifest, or unexplained window parameter, the active corpus must do one of three things:

1. provide a finite support/operator/response derivation in the relevant book;
2. demote the claim to a bridge/comparison protocol in Book 05;
3. remove the statement from the active theorem path and keep it only as historical or diagnostic material.

Protected classes include high powers of `φ`, numerical correction exponents, `Λ_act=38m_ec^2`, meson mass claims, carrier-geometry claims and BAO/S_DE window parameters.

## Primitive Equation Closure Discipline

The base equation is not a naked aesthetic postulate. D0 separates primitive detector obligations from the derived branch equation:

```math
p+p^2=1.
```

The active foundation is the minimal finite positive-response detector skeleton. A cubic return is an iterated runtime with extra hidden memory; a fractional return is not a finite constructive channel composition; a free return weight is a second hidden scale.

## Typed Closure Taxonomy

| status | meaning |
|---|---|
| `CORE-CLOSED` | finite D0 theorem/proof-cell is active in Lean or an owned proof cell |
| `CERT-CLOSED` | finite certificate and negative controls close the computational row |
| `NO-GO` | the current operator set blocks the advertised downstream promotion |
| `BRIDGE-CALIBRATION` | scheme, scale, continuum or response bridge is declared |
| `EMPIRICAL-PASSPORT` | external comparison is declared with data and falsification manifest |
| `EXTERNAL-DATA-REQUIRED` | external files, covariance tables or hashes are absent from the package |

Typed closure prevents a result from being promoted merely because its formula appears in a book or a certificate name exists.

## Synchronization Rule

A Lean theorem that is not integrated into the active books is not part of the scientific strengthening of the corpus. Conversely, a book claim that cannot point to its finite theorem, certificate or no-go owner remains prose.

Current synchronization obligations include Born readout, CKM basis origin, Book 04 strict selectors, Book 04 operator boundaries, finite spin-2 derivation, frozen SM-facing gauge decomposition, internal causal section and the final foundation index.

For gravity, synchronization requires the quotient theorem before Book 07 may speak about a finite spin-2 carrier. For gauge/matter, synchronization requires the frozen finite ledger before Book 04 may speak about an SM-facing gauge decomposition.


The scientific language is finite support, convex response order, coupling kernel, entropy extremal representative and gauge quotient.

## Theory-Improvement Rule

A modification improves the theory only if it removes an unowned primitive, a non-unique selector, an alternative readout not excluded by theorem, a numerical equality without a finite operator, a continuum statement without finite carrier and bridge boundary, or an external comparison that can feed back into the core construction.

Historical closure passes are preserved in the theory map. The active entry contract is theorem/operator/certificate first.
