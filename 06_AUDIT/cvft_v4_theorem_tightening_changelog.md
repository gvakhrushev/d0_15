# CVFT v4 Theorem Tightening Changelog

## A2 Accepted

Boundary rank, log-det rank, UV-tail, `U_eff` pole and S_DE EP algebra were
accepted as theorem-ready targets.

## A2 Corrected

Bounds use `a=|z|rho(F)<1`; log-det uses the analytic branch at `z=0`;
`delta0^12` is a readout threshold.

## B2 Accepted

F1-F6 sector operator designs were integrated as proof obligations or frontier
targets.

## B2 Demoted/Tightened

No alpha derivation, Hawking closure, baryon mass computation, IceCube fix or
Yang-Mills mass-gap solution is claimed.

## New Cert Stubs

The v5 cert set supersedes the v4 stubs while preserving compatibility wrapper
names.

## New Forbidden Patterns

See `cvft_overclaim_audit.md`.

## Remaining Obligations

Exact finite operators, Lean proofs and external passports remain queued.
