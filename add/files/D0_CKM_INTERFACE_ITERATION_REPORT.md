# D0 — Итерация свободного синтеза: сектор смешивания (CKM) из сцены K(9,11,13)

**Статус-дисциплина:** метки по DEF 0.1.1 (GOLDEN). Всё ниже — THE / NO-GO / CHK / HYP, без промежуточных риторических классов.

## 1. THE (структурный): арность матрицы смешивания = 3 угла + 1 фаза

Поколения ≡ межзонные интерфейсы сцены: I₁₂=(9,11), I₂₃=(11,13), I₁₃=(9,13).
Граф интерфейсов (вершины — интерфейсы, смежность — общая зона) есть K₃.

- Число интерфейсов = 3 → **3 угла смешивания**.
- Цикловый ранг K₃ = E − V + 1 = 3 − 3 + 1 = **1** → ровно **одна** неустранимая голономная фаза (CP).
- От противного (схема DEF 0.2.2): четвёртое поколение ⇒ четвёртый интерфейс ⇒ четвёртая зона ⇒ четвёртая структурная необходимость сверх {дефект, память, оболочка}; она операционально различима ⇒ новая роль ⇒ внешний каталог ⇒ ⊥ M1. **Четыре поколения запрещены.**

Это совпадает со структурой CKM (3 угла Кабиббо–Маскавы + δ_CP) и переводит *число* параметров смешивания из «свободных» в вынужденные.

## 3. NO-GO (доказан вычислением): статическая связность не порождает иерархию

Оператор смешивания нормированной связности интерфейсного треугольника
M_ij = shared_zone/√(cap_i·cap_j) даёт {0.0925, 0.1005, 0.0836} —
**плоский** спектр углов (отношения 1 : 1.09 : 0.90) против наблюдаемого
1 : 0.186 : 0.017.

**NO-GO:** иерархия смешивания не выводится из статической топологии сцены ни в
каком ведущем порядке. По M1 единственный оставшийся источник иерархии —
динамическое одевание интерфейсов массовым оператором (φ-градуировка циклов).
Это вынуждает следующий объект теории: оператор смешивания = функция
(масс-лестница × связность), не связность отдельно.

## 4. HYP (предсказательная цель): цепочка d₁₃ → m_s/m_d → угол Кабиббо

Терминальное число ветвей сцены d₁₃ = φ_E(44) = **20** (GOLDEN/v14, THE-уровень).

- Решёточное значение (FLAG): m_s/m_d = **20.0 ± 0.5** — совпадение с d₁₃.
- Канал Гатто–Сартори–Тонина: sinθ_C ≈ √(m_d/m_s).
- **HYP-цель:** sinθ_C = 1/√20 = 1/(2√5) = 1/(2(2φ−1)) = **0.22361** — чисто золотая алгебра.
- Сравнение: 0.22501(68) эксп. → расхождение 0.62% (2.1σ_exp), внутри известной
  точности самого соотношения GST (поправки O(λ²) ~ 5%).
- Честная граница: вторая ступень НЕ обобщается (√(m_s/m_b)=0.139 ≠ 0.042);
  заявляется только первая ступень.

**Программа закрытия HYP→THE:** вывести m_s/m_d = d₁₃ = 20 из φ-лестницы масс
(BOOK III §III.1.C) как отношение чисел циклов на терминальном ветвлении; тогда
угол Кабиббо станет следствием двух THE.

## 5. Побочный HYP (слабый): фаза CP как половина золотого угла

δ_golden/2 = 180°(1−φ⁻¹)·2/2 = 68.75°. Против δ₁₃ = 65.5±1.5° → +2.2σ (натяжка);
против γ = 65.9±3.5° → +0.8σ (совместимо). Статус: слабый HYP, держать в реестре,
не продвигать.

## 6. Сухой остаток итерации

1. THE: арность CKM (3+1) и запрет 4-го поколения — вынуждены сценой.
2. CHK: статическая нумерология углов самоликвидирована замороженным протоколом.
3. NO-GO: статика не даёт иерархии → динамическое одевание вынуждено.
4. HYP: sinθ_C = 1/(2√5) через мост d₁₃=20 ↔ m_s/m_d — цель с программой закрытия.

# Итерация 3: нейтринный сектор — инверсия иерархий, PMNS и шкала масс

## 12. THE (бespараметрический): инверсия иерархий CKM ↔ PMNS

Кварковые токи живут в активном секторе: три невырожденные моды
(21.84, −9.76, −12.08) → иерархия подавляет смешивание (CKM мала).
Нейтрино живут в стерильном архиве: ядро 30-кратно вырождено →
вырождение разрешает любое вращение → смешивание большое (PMNS велика).
**Почему лептоны смешиваются сильно, а кварки слабо — следствие rank-nullity
сцены, ноль параметров.** Это один из главных необъяснённых фактов СМ.

## 13. THE (вычислен): зонная подструктура архива 30 = 8 ⊕ 10 ⊕ 12

Ядро оператора смежности распадается по зонам: внутризонные разностные
векторы дают 8+10+12. Архив наследует трёхзонность сцены → нулевой порядок
PMNS = зонная демократия (тримаксимальный/TBM-паттерн) — внешний «велосипед»
TBM втянут на своё место.

# Итерация 5: кватернионный слой — Q₈ вынужден, спин-1/2 и заварка сустава +2

## 17. THE (новая, вычислена + Дедекинд 1897): Ω₈ ≅ Q₈ — вынуждено, не объявлено

M1-прочтение нормальности: ненормальная подгруппа имеет сопряжённые копии —
различение копии требует каталога — запрещено. Память порядка (COR 3.2.A.1)
требует некоммутативности. Итого: группа ролей обязана быть **гамильтоновой
неабелевой** (все подгруппы нормальны). Полный скан неабелевых групп порядка ≤8:
S₃ — 3 ненормальные подгруппы, D₄ — 4, **Q₈ — ноль**. Теорема Дедекинда (1897):
всякая гамильтонова группа содержит Q₈ → минимальность глобальна, не до порядка 8.
**Ω₈ ≅ Q₈ — теперь теорема.** Декларация 01.7A (первая половина) закрыта.
Втянутый велосипед 130-летней выдержки — магнетизм в действии.

## 18. THE (тройное тождество): память порядка = ориентация = один объект

Вычислено: [Q₈,Q₈] = Z(Q₈) = Φ(Q₈) = {±1}. Коммутант (память порядка операций),
центр (ориентация) и подгруппа Фраттини (остаток-непорождающее) — **один и тот же
ℤ₂**. Структурно это M1-чистейшая из возможных групп: три канонических объекта
без трёх каталогов. Сам ± из Ω₈ — это и есть память о порядке записи/чтения.

## 19. COR: спин-1/2 и трёхмерность

i² = j² = k² = −1: полуоборот в квадрате даёт центральный −1, значит 2π ≠ id,
4π = id — **фермионная статистика = ориентационный бит сцены**. Три мнимые оси —
три направления переноса (согласуется с рангом 3 матрицы смежности).

## 20. LEM-target: заварка мягкого сустава №2 (шаг +2)

Отождествление: «внешний ℤ₂-бит», запрещаемый THE 3.11.B при шаге +1, — это
центральный −1 группы Q₈, индекс листа спинорного двулистного накрытия.
Чётность Люка ε_n = (−1)^{n+1} = номер листа; склейка без бита = ядро
голономии = чётные шаги. Интерпретирующий мостик REM 3.11.B.T получает
алгебраический носитель. **Зазор:** формальное отображение
(группоид склейки слоёв → Z(Q₈)) ещё не выписано — но сустав переведён из
«интерпретация» в «отождествление двух ℤ₂ с путём построения».

## 21. REM (не продвигается)

24 единицы Гурвица (бинарная тетраэдральная 2T ⊃ Q₈); 33 = 24 + 9;

## 22. Сводка сходимости
